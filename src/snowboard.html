Snowboard series

Чехлы для перевозки сноуборд оборудования. Вместительные чехлы с бортом позволяют перевозить сноуборд с креплениями, ботинки и еще что-нибудь. Для перчаток и маски есть отдельный карман.

Основные характеристики и оснащение:

  • Материал чехла Оксфорд 1680
  • Пеноматериал дна и борта 5 мм
  • Крупная пластиковая молния с двумя бегунками
  • Ручки и съёмный плечевой ремень для переноски
  • Внутренние ремни для фиксации доски
  • Карман для аксессуаров с внешним доступом
